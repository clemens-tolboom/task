Installation
------------

0. Download and apply this comment.module patch: http://drupal.org/node/28255
   This module depends on this patch to provide the comment-based
   task updates.

1. Place these files in modules/task:
     task.module
     task.css
     calendar/*

2. Load the database definition using the tool of your choice. For mysql with
   command line access use:
     mysql -u user -p drupal < task.mysql
   Replace user with the MySQL username, and drupal with the table being used.

3. Enable the task module under administer > configuration > modules.

4. Grant the proper access to user accounts under administer > accounts >
   permissions.
